# 🍝 Paparizza — Restaurant Website Demo

A modern, professional single-page restaurant website built with pure HTML, CSS and JavaScript. 
Designed and developed as a demo concept for Paparizza — Trattoria, Pizzeria & Bar, Greater Kailash, New Delhi.

---

## 🌐 Live Demo
https://shiveshcodes-io.github.io/demo-paparizza-website/

---

## ✨ Features

- **Hero Section** — Full-screen background with animated text and Reserve a Table button
- **About Section** — Restaurant story, chef info and key stats with scroll animations
- **Interactive Menu** — Filterable by category (Pastas, Appetizers, Salads, etc.) with food photos and prices
- **Photo Gallery** — Masonry-style grid with hover overlay effects
- **Guest Reviews** — Star ratings and customer testimonials
- **Location** — Embedded Google Maps with address, contact and opening hours
- **WhatsApp Reservation** — Booking form that sends details directly to WhatsApp
- **Responsive Design** — Mobile, tablet and desktop friendly
- **Hamburger Menu** — Clean mobile navigation under 1023px
- **Smooth Animations** — Scroll-triggered reveal animations throughout
- **Scroll to Top** — Fixed button for easy navigation

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables, Animations)
- Vanilla JavaScript (No frameworks, no dependencies)
- Google Fonts — Playfair Display + DM Sans
- Unsplash (placeholder food images)

## 🚀 How to Use

1. Clone or download this repository
2. Open `index.html` in any browser
3. To customize — replace restaurant name, menu items, photos, WhatsApp number and Google Maps link

---

## ⚙️ Customization Guide

| What to change | Where to find it |
|---|---|
| Restaurant name & tagline | Hero section in HTML |
| WhatsApp number | `const WHATSAPP_NUMBER` in JavaScript |
| Menu items & prices | Menu section in HTML |
| Google Maps location | `src` attribute of the iframe |
| Colors | CSS variables at the top (`:root`) |
| Hero background photo | `.hero-bg` in CSS |
